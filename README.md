Ecommerce Sales Dashboard | Power BI

This interactive Ecommerce Sales Dashboard was developed in Microsoft Power BI to provide a comprehensive view of sales performance and customer insights. The dashboard tracks key metrics such as Total Revenue, Total Orders, Average Order Value, Previous Orders, and Average Review Score, enabling businesses to monitor performance at a glance.

It features interactive visualizations that analyze sales by product category, payment method, country, customer traffic source, and review scores, helping stakeholders identify trends, evaluate customer behavior, and make data-driven decisions.

Key Features

📊 KPI cards for Total Revenue, Total Orders, Average Order Value, Previous Orders, and Review Score

💳 Sales analysis by payment method

🛍️ Product category performance analysis

🌍 Sales and customer review analysis by country

👥 Customer age distribution by traffic source

📈 Average order value by product category

🎯 Interactive dashboard for business performance monitoring
